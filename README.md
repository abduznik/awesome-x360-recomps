# Awesome Xbox 360 Recomps

A curated list of awesome Xbox 360 game recompilation projects that bring classic games to modern platforms.

> **What is a recompilation?** Static recompilation translates the original game's Xbox 360 machine code into native code that runs on modern hardware — no emulation needed. The result is a native PC build with improved performance, higher resolutions, and mod support.

---

## Rexglue

Projects built with the [Rexglue SDK](https://github.com/rexglue/rexglue-sdk).

| Game | Description | Author |
|------|-------------|--------|
| [Skate 3](https://github.com/mchughalex/skate3recomp) | Native recompilation of Skate 3 for Windows, Linux, and macOS — runs at 4K 165fps with ultrawide support | [mchughalex](https://github.com/mchughalex) |
| [Naughty Bear](https://github.com/MaxDeadBear/NaughtyBear_ReStuff) | PC port of Naughty Bear (Gold Edition with all DLC) for Windows | [MaxDeadBear](https://github.com/MaxDeadBear) |
| [Lollipop Chainsaw](https://github.com/MaxDeadBear/Re-Cherry) | PC port of Lollipop Chainsaw for Windows | [MaxDeadBear](https://github.com/MaxDeadBear) |
| [Banjo-Kazooie: Nuts & Bolts](https://github.com/masterspike52/reNut) | Recompilation of Banjo-Kazooie: Nuts & Bolts for Windows and Linux, runnable via the Goopie launcher | [masterspike52](https://github.com/masterspike52) |
| [Viva Piñata: Trouble in Paradise](https://github.com/SolarCookies/TiP-Recomp) | Early-stage recompilation of Viva Piñata: Trouble in Paradise with mod support via function hooking, targeting Windows (Linux planned) | [SolarCookies](https://github.com/SolarCookies) |

## XenonRecomp

Projects built with [XenonRecomp](https://github.com/hedge-dev/XenonRecomp) (PowerPC → C++) and [XenosRecomp](https://github.com/hedge-dev/XenosRecomp) (Xenos shaders → HLSL).

| Game | Description | Author |
|------|-------------|--------|
| [Sonic Unleashed](https://github.com/hedge-dev/UnleashedRecomp) | Unofficial PC port of Sonic Unleashed for Windows and Linux — supports high resolutions, ultrawide, high framerates, and mods via Hedge Mod Manager | [hedge-dev](https://github.com/hedge-dev) |
| [Sonic the Hedgehog (2006)](https://github.com/sonicnext-dev/MarathonRecomp) | Unofficial PC port of Sonic '06 for Windows, Linux, and macOS — still in active development | [sonicnext-dev](https://github.com/sonicnext-dev) |

## Contributing

Know of a recomp that should be on this list? **Send a pull request!**

- One project per row
- Include the game name, a short description, and the author
- Place it under the correct engine section (add a new section if needed)
- The project must be publicly available on GitHub or a similar platform
